# RapidPostPe

Aplicación web para la gestión de pedidos de postres.

## Funcionalidades

- Registrar pedidos
- Editar pedidos
- Eliminar pedidos
- Gestionar estados (Pendiente, Debe, Entregado)
- Registrar abonos
- Calcular deuda automáticamente
- Exportar pedidos a CSV
- Guardar información en LocalStorage

## Tecnologías

- HTML
- CSS
- JavaScript
